## Build Instructions

1. You must install Node.js and Yarn before following the steps below.

2. Download or clone the repository.

3. Go to the project's root directory and install dependencies.

```zsh
% npm install
```

4. Happy developing

```zsh
# Development
% npm run dev
```

```zsh
# Build and start the server
% npm run build
% npm run start
```
